## 7. 結論

1. **直送区分91が標準**（70.3%）で、通常の倉庫納品を示す
2. **直送区分11は店舗直送**（14.2%）
3. **倉庫0091は直送専用**で、20番台の直送区分を使用
4. **分類コード末尾0は直送対象**、末尾1以降は倉庫納品のみ
5. **Ｍ１店舗に存在しない直送区分**（20, 24-29, 64）は直送先コードとして別管理が必要
