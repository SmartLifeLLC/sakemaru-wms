このシステムの解説書が必要。 メガメニューの解説の発注の部分に内容を追加したい。

メニューの場所
- メガメニュー -> 解説 -> 発注 -> 発注の流れ

発注の流れの説明
1. 朝9:30分 HUB倉庫締切の早い仕入れ先の発注を実施
2. 朝11:30分まで　各サテライト倉庫の発注を実施
3. 12:00 まで倉庫の発注を実施

- 注意点
1. オレンジ倉庫は外部倉庫ではあるが、内部のHUB倉庫と同じ動きになる。
- サテライト倉庫はオレンジ倉庫に在庫移動依頼。
- オレンジ倉庫は在庫移動のための波動出荷が必要
- ピッキングリストを出力し、オレンジ倉庫に出荷指示を実施
- ピッキングをウェブ上で実施（実際の在庫分の移動）
- オレンジ倉庫の在庫管理が必要
2. サテライト倉庫の移動発注が実施されなくてもHUB倉庫の発注は可能。その際に移動出荷分は考慮されない。
